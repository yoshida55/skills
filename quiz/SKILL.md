# quiz スキル

## 使い方
```
/quiz
```


---

## 📋 Step 2. 出題優先順位

ファイルを読んで以下の順で出題する：

| マーク | 意味 | 出題頻度 |
|---|---|---|
| ❌ | 不正解（要復習） | 毎回出す |
| なし | 未出題 | 毎回出す |
| ✅ | 正解（定着済み） | 5問に1回 |

---

## 🎯 Step 3. 出題ルール

- **1問ずつ出す**（まとめて出さない）
- **今日の日付セクション（`## YYYY-MM-DD` が今日）の項目は出題しない**（当日追加分はスキップ）
- **同じ会話内で既に出題した問題は出題しない**（短期記憶対策）
- 問題形式は内容に合わせて選ぶ：
  - 穴埋め：`get_the_category() は〇〇で返る`
  - 選択：`A: echo必要 / B: 不要`
  - コード補完：`このコードの〇〇に入るものは？`
- ヒントは求められたとき or 明らかに詰まっているときだけ出す

---

## ✅ Step 4. 採点・記録ルール

### 正解の場合
- 対象行の先頭に `✅` を付ける（既に `✅` なら変えない）
- 「正解です！✅」と伝えて次の問題へ

### 不正解の場合
- 対象行の先頭に `❌` を付ける（既に `❌` なら変えない）
- 正解を解説して次の問題へ

### マークの付け方（Edit ツールで既存行を書き換える）
```
- get_the_category() は配列で返る
↓ 正解なら
- ✅ get_the_category() は配列で返る
↓ 不正解なら
- ❌ get_the_category() は配列で返る
```

---

## 📊 Step 5. スコア表示

各問題の後にスコアを表示する：
```
スコア：✅✅❌✅ 3/4問
```

区切りは**10問ごと**に結果サマリーを出す：
```
## 結果（10問）
✅ 8問正解 / ❌ 2問要復習

❌ 要復習リスト：
- esc_html / esc_url / esc_attr の使い分け
- CF7 の pタグ削除
```

---

## ⚠ 注意事項

- `✅` が付いている項目は基本的にスキップ（5問に1回だけ出す）
- マークは **Edit ツール** で付ける（memo_write は追記専用なので使わない）
- 問題は **1問ずつ** 出す。まとめて出さない

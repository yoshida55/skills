# progress スキル

現在の学習進捗をメモファイルの既存セクションに上書き更新するスキル

## 使い方
```
/progress
```

---

## 実行フロー

### Step 1: PC判定（1会話に1回だけ）
```bash
test -f "D:\50_knowledge\01_memo.md" && echo "home" || echo "company"
```
- 自宅PC → `D:\50_knowledge\01_memo.md`
- 会社PC → `C:\Users\guest04\Desktop\高橋研三\03_knowledge\01_memo.md`

### Step 2: 既存セクションを探す
Grep で `## 学習進捗` を検索する
- ✅ あれば → その行から次の `##` までを**上書き**
- ❌ なければ → ファイル末尾に新規追加

### Step 3: 内容を更新する
会話の内容から進捗を自動判断して書く

---

## 保存フォーマット

```markdown
## 学習進捗

### 勉強中
- （例: PHP/WordPress テーマ開発）

### ✅ 完了
- 完了した項目を箇条書き

### ➡ 次にやること
- 次のタスク（1〜3個まで）

### 📅 最終更新
- YYYY-MM-DD
```

---

## ルール

- 毎回**新規追加しない** → 既存の `## 学習進捗` セクションを上書き
- 「次にやること」は1〜3個まで（多すぎない）
- 完了したものは「✅ 完了」に移す
- シンプルに保つ（長文にしない）
- memo-format と同じ場所チェックを使う
